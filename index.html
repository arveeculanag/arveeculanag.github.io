Projects

Throughout my extensive career as a developer, I have spearheaded the development of several notable projects, each showcasing my proficiency and expertise in the field. Below, I present a curated selection of some of my most significant contributions to date..

Arvee's Restaurant (GitHub)

Incorporating skills from "The Origins I: HTML" course, I developed a gesture-controlled music player using React. Applying semantic HTML, creating components, handling props and states, and implementing form validations, I created an interactive musical experience. This project demonstrates my basic skills in React and my ability to creatively apply learned principles.

Guess That Compound

Developed an Android quiz game application designed to challenge users to identify chemical compounds within a specified time frame. Implemented a dynamic interface displaying compound names for user interaction. Incorporated a "Learn" feature enabling users to access concise daily lessons, fostering continuous learning in byte-sized modules.

PyMusical

This project seamlessly integrates my two passions: music and programming. Utilizing hand gestures and a camera, the system translates gestures into musical notes, offering a unique and interactive music-playing experience. Each hand gesture corresponds to a specific musical note, enhancing user engagement and creativity. This project showcases my proficiency in programming and innovative thinking, combining technical skills with artistic expression.
